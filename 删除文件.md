### 删除文件

- 一般就直接在工作区删除，或使用**rm 小贴士.txt**。这个时候，git知道你删除了文件。此时工作区和版本库就不一样了，使用**git status**会立刻告诉你哪些文件被删除。那么现在就有两种情况：

  - 你确实要从版本库中也删除，就用

    > $ git rm 小贴士.txt

    然后再commit

    > $ git commit -m "删除小贴士"

    现在工作区和版本库都删除了。

  - 删错了，这是不用担心，因为库里还有，可以用库里的替换工作区的,即一键还原，**但要注意只能恢复到最新版本。**

    > git checkout -- 小贴士.txt
